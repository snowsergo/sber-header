# Sber-header
Тестовое задание адаптивный header

## Текущая версия
1.0.0


## Описание работы:
Динамически изменяющийся header и выпадающее меню

## Установка проекта

Скопируйте проект на компьютер:
```
git clone https://github.com/snowsergo/sber-header.git
```
Установите зависимости проекта:
```
npm i
```
Запуск в dev режиме:
```
npm run dev
```
Финальная сборка:
```
npm run build
```
